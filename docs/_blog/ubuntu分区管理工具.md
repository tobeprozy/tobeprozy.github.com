第一步：打开分区管理工具  
使用命令安装分区管理工具[gparted](https://so.csdn.net/so/search?q=gparted&spm=1001.2101.3001.7020):

```cpp
sudo apt-get install gparted
```
  
使用命令启动分区管理工具

```cpp
sudo gparted
```
